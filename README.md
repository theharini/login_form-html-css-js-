# Login Form - HTML, CSS & JavaScript

A simple and responsive **Login Form** built using **HTML, CSS, and JavaScript**.  
This project demonstrates basic frontend development skills including form design, validation, and responsive UI.

---

## 🚀 Features

- Clean and modern UI
- Responsive design
- Form validation using JavaScript
- Password input field
- User-friendly layout

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
